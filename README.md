# ATM System (C++)

A console-based ATM System built in C++ that allows clients to log in, perform withdrawals, deposit money, and check account balances using file-based data storage.

### Features

- Client login system
- Quick withdrawal operations
- Normal withdrawal operations
- Deposit functionality
- Balance inquiry
- File-based data persistence

### Concepts Practiced

- File Handling in C++
- Authentication Basics
- CRUD-Style Data Updates
- Functions and Modular Programming
- Business Logic Implementation
- Financial Transaction Processing

### Technologies

- C++
- Console Application
- File System Storage

## Purpose

This project was developed to practice implementing a simple banking workflow, including authentication, account management, and transaction processing using file-based persistence.
